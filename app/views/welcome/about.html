This website arose after experiencing a series of delays and cancellations to train services. I noticed that some train services didn't experience delays/cancellations, while others did regurarly.

After doing some research, I found that National Rail records the arrival performance of each train service and makes this information accessible by an API. This website uses this API and displays this information in a readable format. It aims to help people decide which train service to take to reach a destination on time.

For each train service in a given hour, the website shows one of the following badges:

  • Great - this train service has an on time arrival performance of 90% or more.
  • OK - this train service has an on time arrival performance of 80% or more.
  • Avoid - this train service has an on time arrival performance of less than 80%.

The "on time" definition can be adjusted by setting the "Arrival allowance" field, which will change the number of minutes that a train service must arrive within its set arrival time.
